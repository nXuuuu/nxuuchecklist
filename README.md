# nXuu CheckList

A clean, minimal pre-trade checklist for disciplined trading execution — built to be used before entering any trade.

**Live site:** `https://<your-username>.github.io/nxuu-checklist`

---

## Features

- 10-step structured checklist across 5 phases
- Smooth progress bar
- Fully responsive — works on desktop, tablet, and mobile
- Touch-friendly (tested on iOS Safari & Android Chrome)
- Keyboard accessible
- No dependencies — pure HTML, CSS, JavaScript

## How to Deploy on GitHub Pages

1. Create a new repository on GitHub named `nxuu-checklist`
2. Upload all files (keep the folder structure)
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch → `/ (root)`
5. Click **Save**
6. Your site will be live at `https://<your-username>.github.io/nxuu-checklist`

---

## Checklist Phases

| # | Phase | Steps |
|---|-------|-------|
| 01 | HTF Context | Swing Point, PD Array / POI |
| 02 | Liquidity | Session Liquidity, LQ Sweep |
| 03 | Confirmation | Candle closes in zone, MSS / CHoCH |
| 04 | Entry | LTF PD Array, FVG / OB / IFVG / CISD |
| 05 | Trade Management | RR ≥ 1:2, Risk ≤ 1% |
